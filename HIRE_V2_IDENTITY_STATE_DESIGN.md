# 一、研究问题

v16.2.1 已经证明，同身份不同图像构成的严格留一组共识可以为完整图文观测提供有效的身份稳定残差。其最佳检查点约为：

```text
R1：58.034
R5：76.394
R10：82.998
mAP：44.540
mINP：24.503
```

身份残差相对同一检查点的完整观测约提高：

```text
R1：+0.539
mAP：+0.491
mINP：+0.508
```

但身份分支单独检索的 R1 只有约 `52.902`。这说明身份分支最适合补充同身份跨图公共因素，不适合作为当前图文细节判断的主体。

因此，v16.3.0 不再修改身份监督，而只回答一个问题：

> 当身份组共识已经把正确身份召回到前列后，能否利用当前文本中的局部词语和当前候选图像中的局部区域，修复相似身份之间的第一名判断？

状态匹配必须和身份匹配具有不同的统计含义。

身份级匹配回答：

```text
这条文本和这张图像是否共享同一个人的跨图公共因素？
```

状态级匹配回答：

```text
这条文本中当前被描述的局部细节，是否在这张候选图像中得到支持？
```

因此，状态分支不能继续做成另一个独立全局向量头，也不能使用同身份支持包把不同图像的状态无条件传播。

\[

(l_i^T)^\top l_j^I.

\[
o_i^I
=
\operatorname{Norm}
(b_i^I+e_i^I),

=
\operatorname{Norm}
(b_i^T+e_i^T).

\[

# 五、状态分支为什么必须是成对局部匹配

若状态分支仍然采用：

```text
完整观测
→ 线性层
→ 状态全局向量
```

它仍然可能复制身份空间，无法保证关注当前可见细节。

v16.3.0 改为成对晚交互：

```text
文本先保留多个局部词元；
图像先保留多个局部图像块；
只有在给定文本和给定候选图像后，
才计算状态兼容分数。
```

因此，状态分数不能提前压缩成一个独立全图库检索向量。

该结构天然更适合判断：

```text
鞋子颜色是否对应；
是否携带背包；
是否戴帽；
上衣或下装局部是否对应；
当前姿态是否符合；
某个细节是否在候选图像中存在清晰局部响应。
```

## 6.1 图像状态块

设图像最后一层词元为：

\[
X_i^I
=
[x_{i,\mathrm{cls}}^I,
x_{i,1}^I,\ldots,x_{i,N}^I].
\]

使用类别词元到图像块的最后一层注意力：

\[
a_{i,n}^{I}.
\]

排除类别词元，选择注意力最大的十六个图像块：

\[
\mathcal V_i
=
\operatorname{Top16}
\left(
\{a_{i,n}^{I}\}_{n=1}^{N}
\right).
\]

## 6.2 文本状态词元

设文本结束词元位置为 \(e_i\)。

使用结束词元到有效文本词元的最后一层注意力：

\[
a_{i,m}^{T}.
\]

排除：

```text
开始词元；
结束词元；
填充词元。
```

选择注意力最大的八个有效词元：

\[
\mathcal W_i
=
\operatorname{Top8}
\left(
\{a_{i,m}^{T}\}
\right).
\]

若实际有效词元不足八个，其余位置通过掩码完全屏蔽。

选中词元的重要度为：

\[
w_{i,m}
=
\frac{
\exp(a_{i,m}^{T})
}{
\sum_{r\in\mathcal W_i}
\exp(a_{i,r}^{T})
}.
\]

无效位置权重严格为零。

# 七、共享状态词元投影

图像状态块和文本状态词元共用同一个无偏置线性投影：

\[
W_s
\in
\mathbb R^{128\times512}.
\]

投影后：

\[
q_{i,m}
=
\operatorname{Norm}
(W_s x_{i,m}^{T}),
\]

\[
v_{j,n}
=
\operatorname{Norm}
(W_s x_{j,n}^{I}).
\]

图像和文本共享投影的原因是：

```text
CLIP 词元已经处于跨模态语义空间；
共享投影避免两个模态任意旋转；
状态分支学习的是共同局部兼容坐标，而不是两个独立分类空间。
```

\(W_s\) 使用行正交初始化。

## 7.1 梯度保护

送入状态投影前的图像和文本原始词元全部停止梯度：

\[
q
=
W_s\operatorname{sg}(x^T),
\]

\[
v
=
W_s\operatorname{sg}(x^I).
\]

因此：

```text
状态辅助损失不能直接修改 CLIP 主干；
状态辅助损失不能修改 RDE 局部分支；
状态辅助损失不能修改完整观测；
状态辅助损失不能修改身份映射；
状态辅助损失只训练状态词元投影。
```

完整观测和身份基础仍然通过原有检索主损失端到端更新。

# 八、状态局部晚交互分数

给定第 \(i\) 条文本和第 \(j\) 张候选图像，词元—图像块相似度为：

\[
A_{ijmn}
=
q_{i,m}^{\top}v_{j,n}.
\]

对于每个文本状态词元，寻找候选图像中最匹配的局部块：

\[
c_{ijm}
=
\max_{n\in\mathcal V_j}
A_{ijmn}.
\]

最终状态兼容分数为：

\[
S_{ij}^{S}
=
\frac{
\sum_{m\in\mathcal W_i}
w_{i,m}c_{ijm}
}{
\sum_{m\in\mathcal W_i}w_{i,m}
}.
\]

因为有效文本权重已经归一化，分母通常等于一。

该分数具有明确解释：

```text
文本中的每个重要局部词语，
必须在候选图像的某个重要局部区域找到最佳响应；
文本词元对图像的贡献由原 CLIP 结束词元注意力决定；
最终状态分数是当前文本条件下的候选图像局部兼容度。
```

状态分数不是独立全局身份分数。

状态晚交互比普通向量点积开销大，因此状态只处理身份基础已经召回的高相似候选。

训练中根据停止梯度后的身份平衡分数：

\[

为每条文本选择前：

\[

个图像候选。

这一强制只发生在训练关系构造中。

# 十、状态正负关系

状态关系由 `image_id` 和身份编号共同定义。

## 10.1 状态正样本

\[
P_{ij}^{S}
=
\mathbf1[g_i=g_j].
\]

即：

```text
当前图像；
以及同一 image_id 对应的另一条文本；
```

属于状态正关系。

这修复了普通对角对比损失中“同一张图的第二条文本可能成为假负样本”的问题。

## 10.2 状态负样本

\[
N_{ij}^{S}
=
\mathbf1[p_i\neq p_j].
\]

不同身份属于状态负关系。

## 10.3 状态忽略样本

满足：

\[
p_i=p_j,
\qquad
g_i\neq g_j
\]

的同身份不同图像既不作为状态正样本，也不作为状态负样本。

原因是：

```text
它们共享身份；
但当前姿态、视角、遮挡和局部可见细节不一定相同；
不能无条件拉近；
也不能当作错误状态强行推远。
```

# 十一、状态配对损失

候选掩码记为：

\[
M_{ij}^{C}.
\]

文本到图像方向的允许集合为：

\[
A_{ij}^{t2i}
=
M_{ij}^{C}
\land
(P_{ij}^{S}\lor N_{ij}^{S}).
\]

多正样本状态 NCE：

\[
L_{\mathrm{state}}^{t2i}
=
-\frac1{|\mathcal Q|}
\sum_{i\in\mathcal Q}
\log
\frac{
\sum_{j:
M_{ij}^{C}\land P_{ij}^{S}}
\exp(sS_{ij}^{S})
}{
\sum_{j:
M_{ij}^{C}\land(P_{ij}^{S}\lor N_{ij}^{S})}
\exp(sS_{ij}^{S})
}.
\]

图像到文本方向使用转置后的：

```text
状态分数；
状态正负掩码；
候选掩码。
```

最终状态辅助：

\[
L_{\mathrm{state}}
=
\frac12
\left(
L_{\mathrm{state}}^{t2i}
+
L_{\mathrm{state}}^{i2t}
\right).
\]

状态辅助权重复用身份组辅助系数：

\[
\lambda=0.1.

因此没有新增第四个损失权重超参数。

# 十二、零初始化状态残差

状态门定义为：

\[
\beta
=
\tanh(\theta_s).
\]

初始化：

\[
\theta_s=0.
\]

因此：

\[
\beta=0.
\]

并且：

\[
\frac{\partial\beta}
{\partial\theta_s}
\bigg|_{\theta_s=0}
=1.
\]

状态门既满足训练开始时精确为零，又不会出现零梯度。

状态分支已经通过 \(L_{\mathrm{state}}\) 学习“高分表示当前图文局部兼容”，因此有效训练后预期：

\[
\beta>0.
\]

若最终 \(\beta\leq0\)，需要视为状态机制没有形成预期语义。

# 十三、最终状态残差分数

状态只作用于前五十候选。

对于候选位置：

\[
S_{ij}^{F}
=
S_{ij}^{B}
+
\beta
\left[
S_{ij}^{S}
-
\operatorname{sg}(S_{ij}^{B})
\right].
\]

数值上，当 \(\beta>0\) 时等价于：

\[
S_{ij}^{F}
=
(1-\beta)S_{ij}^{B}
+
\beta S_{ij}^{S}.
\]

对于非候选位置：

\[
S_{ij}^{F}
=
S_{ij}^{B}.
\]

状态残差的含义是：

```text

状态分数提供当前文本—候选图像的局部兼容；
状态门决定局部状态证据应替代多少身份基础分数。
```

0.1L_{\mathrm{group}}
+
0.1L_{\mathrm{state}}.
\]

# 十五、为什么主损失这样分配

v16.2.1 的主检索部分是：

\[
0.5L_O+0.5L_B.
\]

v16.3.0 将原来的 \(0.5L_B\) 平分为：

\[
0.25L_B+0.25L_F.
\]

训练开始时：

\[
\beta=0,
\]

所以：

\[
S^F=S^B,
\qquad
L_F=L_B.
\]

于是：

\[
0.5L_O
+
0.25L_B
+
0.25L_F
=
0.5L_O+0.5L_B.
\]

因此 v16.3.0 在训练第一步的主检索函数值严格等于 v16.2.1。

同时，代码审计验证初始梯度也满足：

\[
\nabla
\left[
0.5L_O+0.25L_B+0.25L_F
\right]
=
\nabla
\left[
0.5L_O+0.5L_B
\right].
\]

这保证新增状态分支不会在初始化时改变已经有效的身份基础训练强度。

# 十六、代码中的损失返回

训练器只会求和名称中包含 `loss` 的返回值。

v16.3.0 返回四个损失项：

```text
sdm_loss
itc_loss
identity_group_loss
state_pair_loss
```

其中：

```text
sdm_loss
= 0.5 × (global_sdm + local_sdm)
+ 0.5 × observation_sdm
+ 0.25 × identity_final_sdm
+ 0.25 × state_final_sdm

itc_loss
= 0.5 × (global_itc + local_itc)
+ 0.5 × observation_itc
+ 0.25 × identity_final_itc
+ 0.25 × state_final_itc

identity_group_loss
= 0.1 × identity_group_nce

state_pair_loss
= 0.1 × state_pair_nce
```

# 十七、单批次训练流程

每个训练批次严格执行：

1. 通过 CLIP 编码主批次图像和文本；
2. 取得图像与文本最后一层注意力；
3. 生成全局观测；
4. 生成 RDE 风格池化局部观测；
5. 生成完整观测；
6. 计算全局和局部锚点目标；
7. 计算完整观测目标；
8. 对停止梯度后的完整观测应用共享身份映射；
9. 编码动态同身份支持图；
10. 构造严格留一身份组简单共识；
11. 计算身份组 NCE；
12. 计算 v16.2.1 身份平衡分数；
13. 计算身份平衡分数目标；
14. 从停止梯度后的身份平衡分数选择前五十候选；
15. 强制将同 image_id 状态正样本加入训练候选；
16. 从停止梯度的 CLIP 原始词元中选择八个文本词元和十六个图像块；
17. 经过共享状态投影；
18. 计算候选级文本词元—图像块 MaxSim；
19. 计算状态配对 NCE；
20. 通过零初始化状态门构造状态最终分数；
21. 计算状态最终分数目标；
22. 按固定公式聚合总损失；
23. 一次反向传播和一次优化器更新。

# 十九、计算开销

主干和身份支持路径与 v16.2.1 相同。

新增状态训练计算大约只涉及：

```text
64 条文本；
每条前 50 个候选；
每条文本最多 8 个词元；
每张图最多 16 个图像块；
状态维度 128。
```

成对相似度规模为：

\[
64\times50\times8\times16.
\]

测试只对前五十候选做相同计算，而不是对全部 8,252 张图库图像做晚交互。

状态词元缓存大致为：

```text
文本：16504 × 8 × 128；
图像：8252 × 16 × 128；
```

可在普通服务器内存中保存。

# 二十四、失败判定与后续决策

若出现以下任一情况：

```text
state_gate 接近 0；
状态正负间隔不形成；
state_final 低于 identity_final；
破坏数多于修复数；
mAP 明显下降；
```

则说明当前无标签局部状态晚交互没有形成有效互补。

此时不立即加入：

```text
MLLM 标签；
视角监督；
三态标签；
困难负样本；
更多状态层。
```

应先利用本版本输出的候选级状态分数和修复/破坏案例，判断问题来自：

```text
文本词元选择；
图像块选择；
MaxSim 聚合；
状态关系标签；
状态门方向；
候选覆盖率。
```

若 v16.3.0 有稳定增益，则为：


补随机种子二和三。

# 二十六、最终方法解释

v16.3.0 的完整层级可以写为：

\[
\text{完整观测固定效应}
\rightarrow
\text{身份组共识残差}
\rightarrow
\text{当前状态局部残差}.
\]

最终匹配为：

\[
S^{F}
=
S^{O}
+
\Delta S^{I}
+
\Delta S^{S}.
\]

其中：

```text
完整观测：
描述图像和文本的全局、局部混合语义。

身份残差：
利用同身份不同图像的留一组共识，
补充跨图稳定的身份公共因素。

状态残差：
仅在身份基础的高相似候选中，
利用当前文本词元和当前候选图像块的局部兼容关系修正排序。
```

状态分支不传播同身份其他图像的当前细节，也不把不可确认的同身份图像视为状态正样本。

这使身份级匹配和状态级匹配在：

```text
输入；
正样本关系；
表示形式；
计算位置；
推理职责；
```

上都具有实质区别，而不是两个名称不同的全局投影头。
